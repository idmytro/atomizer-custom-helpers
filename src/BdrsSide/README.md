# `BdrsSide`

```css
.Bdrs<X-side>($0) {
  border-top-<X-side>-radius: $0;
  border-bottom-<X-side>-radius: $0;
}

.Bdrs<Y-side>($0) {
  border-<Y-side>-left-radius: $0;
  border-<Y-side>-right-radius: $0;
}
```

## Example

```html
<div class="BrdsT(1rem)"></div>
<div class="BrdsStart(1rem)"></div>
```
